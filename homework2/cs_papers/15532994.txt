Title: Modeling input files for "Supershear rupture of the 1995 Mw7.2 multi-segement Nuweiba earthquake in the Gulf of Aqaba"

Authors: Li, Bo, Mai, Paul Martin

Keywords: 

Publication date: 2025-05-28

Description:

Using the input files provided, we successfully reproduce the 1995 Mw 7.2 Nuweiba earthquake in the Gulf of Aqaba with the open-source seismic simulation code SeisSol (https://github.com/SeisSol/SeisSol).
